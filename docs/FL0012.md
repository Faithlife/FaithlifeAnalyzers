# FL0012: `Command` should not be used with an interpolated string; use `CommandFormat` instead

[Faithlife.Data](https://github.com/Faithlife/FaithlifeData) has special support for [formatted SQL](https://faithlife.github.io/FaithlifeData/#formatted-sql), which should be used instead of passing a normal interpolated string to `DbConnector.Command`. Consider using `DbConnector.CommandFormat` instead.

Invalid:

```csharp
var whereSql = visible ? "" : " where not deleted";
return connector.Command($"select count(*) from widgets{whereSql};").Query<long>();
```

Valid:

```csharp
var whereSql = visible ? Sql.Empty : Sql.Raw(" where not deleted");
return connector.CommandFormat($"select count(*) from widgets{whereSql};").Query<long>();
```
