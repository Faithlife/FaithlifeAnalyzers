# FL0021: Use null propagation for `Nullable<T>.HasValue ? Nullable<T>.Value : null`

Instead of using the conditional operator:

```csharp
Nullable<T> nullable = ...;
var result = nullable.HasValue ? nullable.Value.Property : null;
```

Use the null-conditional operator:

```csharp
Nullable<T> nullable = ...;
var result = nullable?.Property;
```
